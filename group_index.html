Dason

The fourth stylesheet (Dason's) on this document is based on the idea of resizing the page, and of keeping a specific desired layout as it does. If you zoom in, the page should adjust its layout once the screen is zoomed in a certain amount. The original attempt was to create a resizing element, but that proved difficult. This css demonstrates the use of separate div classes and a couple of id's as well as the background+color properties. The height of the columns are attempting to match one another, but proves difficult to get the column alignments to match up...
